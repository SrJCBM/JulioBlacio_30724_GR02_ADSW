Evidencia tecnica del prototipo

MVC por capas

Las vistas HTML/JS consumen controladores PHP, que delegan reglas a servicios y persistencia a DAO.

View -> Controller -> Service -> DAO

Builder

Construccion validada de entidades complejas como Usuario, Clase, Membresia, Progreso y Reporte.

Service -> Builder -> Entidad valida

DAO / Repository

El SQL se mantiene aislado en repositorios con PDO sobre MySQL, listo para despliegue.

DAO -> PDO -> MySQL

Sesion y roles

AuthController y Auth.php validan login, estado activo y permisos; nav.js solo renderiza el menu.

correo + contrasena -> rol
